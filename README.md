# Тестовое задание Loan App

## Описание проекта

(Общее время выполнения: ~6 часов)

Микросервис для обработки заявок на займы

**Технологии**: PHP 8.4, Postgres 17.0, Nginx, Docker

**API Endpoints:**

- POST /requests - Создание заявки на займ
- GET /processor?delay=5 - Обработка заявок

## Установка

1. Создать файл `.env`  в корне проекта по примеру файла `.env.example`

```
cp .env.example .env
```

2. По необходимости, сконфигурировать `.env` файл (все переменные работают из "коробки")

3. Запустить `start.sh`

```
./start.sh
или
/bin/bash start.sh
```

4. Приложение работает по адресу `localhost:80`

```
Healthcheck - localhost:80/health
```
